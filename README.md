## Objetivo

Criar um site de documentacao da linguagem Go utilizando Zensical, Markdown, GitHub Actions e GitHub Pages.

O projeto tambem teve como objetivo aplicar um fluxo colaborativo profissional com branches, Pull Requests, revisoes de codigo e automacao de CI/CD.

## Tecnologias utilizadas

- Go
- Markdown
- Zensical
- Python
- Git
- GitHub
- GitHub Actions
- GitHub Pages

## Estrutura da documentacao

A documentacao criada aborda os seguintes temas:

1. Introducao e Instalacao
2. Sintaxe Basica e Variaveis
3. Estruturas de Controle
4. Arrays, Slices e Maps
5. Structs e Metodos
6. Tratamento de Erros
7. Goroutines
8. Channels
9. Go Modules
10. Testes Automatizados em Go
