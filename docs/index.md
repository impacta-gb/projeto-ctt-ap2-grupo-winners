# Documentacao da Linguagem Go

Bem-vindo ao site de documentacao da linguagem Go desenvolvido para a AP2 da disciplina CTT.

Este projeto foi criado com Zensical, Markdown, GitHub Actions e GitHub Pages, seguindo um fluxo colaborativo com branches, Pull Requests, revisoes e automacao de CI/CD.

## Objetivo do projeto

O objetivo principal e apresentar uma documentacao introdutoria sobre a linguagem Go, explicando seus principais conceitos com exemplos praticos.

## Conteudos abordados

| Pagina | Tema |
| --- | --- |
| Introducao e Instalacao | Visao geral da linguagem e primeiros passos |
| Sintaxe Basica e Variaveis | Estrutura inicial, variaveis e tipos |
| Estruturas de Controle | If, for e switch |
| Arrays, Slices e Maps | Colecoes em Go |
| Structs e Metodos | Tipos personalizados e metodos |
| Tratamento de Erros | Uso de error, nil e err != nil |
| Goroutines | Concorrencia com goroutines |
| Channels | Comunicacao entre goroutines |
| Go Modules | Gerenciamento de pacotes |
| Testes Automatizados | Testes com o pacote testing |

!!! tip "Navegacao"

    Use o menu lateral para acessar cada topico da documentacao.
